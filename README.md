C++ Basics Collection

This repository contains a collection of basic C++ programs designed to practice fundamental programming concepts.

📚 Topics Covered

- Input and Output (cin / cout)
- Conditional Statements (if / else)
- Arithmetic Operations
- Even and Odd Numbers
- Positive, Negative, and Zero detection

🧩 Example Programs

- Check if a number is positive, negative, or zero
- Check if a number is even or odd
- Basic calculator (addition, subtraction, etc.)

🎯 Purpose

The goal of this repository is to build a strong foundation in C++ by implementing simple programs from scratch and understanding core logic.

🚀 How to Run

Compile any file using:
g++ filename.cpp -o program

Run:
./program

📌 Notes

All programs are written for learning purposes and will be improved over time.
